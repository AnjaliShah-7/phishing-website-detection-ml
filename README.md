
#  Phishing-Website-Detection-ml

An end-to-end Machine Learning project to detect whether a website is **Legitimate** or **Phishing** using various URL and website-related features.

This project follows a modular, production-ready architecture and is being developed using Python and Machine Learning best practices.

---

##  Project Overview

Phishing websites are designed to steal sensitive user information such as usernames, passwords, and banking details by impersonating trusted websites.

This project aims to build a machine learning model that can accurately classify a website as **Phishing** or **Legitimate**.
